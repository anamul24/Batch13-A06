# Digitools-Platform #
## Project Overview
--Digitools is a modern web application where users can explore and purchase premium digital tools, such as AI writing software, design templates, stock assets, etc.

### Description
-- This project is built to simplify access to high-quality digital resources for creators, developers, and marketers. Users can browse curated tools, add them to the cart, and simulate a checkout process with a  clean and responsive ui

## Technologies Used
   1- React.js
   2- Vite
   3- Tailwind CSS
   4- React Toastify

## Key Feaures
 1-Smart Cart System
   i. added the product to the cart
   ii. prevent duplicate items
   iii. remove items easily
   iv. dynamic total price calculation
   
2-Interactive Notifications
  i. real time toast notification
  ii. success, error, and info alert

3- Responsive & Modern Ui
  i. Fully responsive design
  ii. Mobile-friendly navigation
